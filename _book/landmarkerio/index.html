landmarker.io


landmarker.io is a web app for 3D mesh and image annotation using HTML5 in the browser.

The app is live at https://www.landmarker.io and we suggest you go there first to try out the demo mode. If you would like to learn more about mass annotations, we suggest you head over to the Github Page.
